# Contributing to og-simulation-ue

DCO sign-off (`Signed-off-by:` line in commit messages) is appreciated but not enforced.

## Pull requests

- Open an issue first for non-trivial changes.
- Keep PRs focused: one logical change per PR.
- For changes to simulation logic, make them in [og-simulation](https://github.com/og-framework/og-simulation) and bump the submodule pin here.
- Add `// SPDX-License-Identifier: MPL-2.0` to any new source files you author.

## Issues

Please include a minimal reproduction and platform/UE version when reporting bugs.
